How does this work?

  1. Once a week, the raw data from UVHFS is ingested (see notice here for info regarding why) -

  2. Any repeaters marked as offline are not ingested (does not apply to temprarily offline)

  3. Data is presented with Closed repeaters filtered out by default

Notes

Multi-band filtering: The application includes the ability to filter by multiple bands. Use CMD/Ctrl or Shift.

Links: In order for a link to be shown, bidrectional validation must pass. In other words, if repeater A references repeater B as a link, repeater B must then also reference repeater A. If this criteria is met, you'll see the link shown in 'Draw Other Links' ('other', in this case meaning 'non-IMIT').
There is not a great way to handle repeaters that fail bidirectional validation unless there is only a single repeater on the frequency that said repeater specifies as its' link. The 'Draw Non-Validated Links' button allows you to see such links on the map.

Cactus Intertie: Becuase the Cactus Intertie is a closed system, filters for 'Draw Other Links' and 'Show Closed Repeaters' must be enabled.
